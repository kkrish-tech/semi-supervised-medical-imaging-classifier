# COVID-19 Chest X-Ray Database

A team of researchers from Qatar University (Doha, Qatar), the University of Dhaka (Bangladesh), along with collaborators from Pakistan and Malaysia, in collaboration with medical doctors, has created a database of chest X-ray (CXR) images containing COVID-19 positive cases, normal cases, and viral pneumonia cases.

This dataset is released in multiple stages:

- **First release:** 219 COVID-19, 1,341 normal, and 1,345 viral pneumonia CXR images  
- **First update:** COVID-19 class expanded to 1,200 CXR images  
- **Second update:** Dataset expanded to 3,616 COVID-19 positive cases, 10,192 normal cases, 6,012 lung opacity (non-COVID lung infection) cases, and 1,345 viral pneumonia cases, along with corresponding lung masks  

The dataset will continue to be updated as new COVID-19 pneumonia X-ray images become available.

---

## COVID-19 Data Sources

COVID-19 images are collected from multiple publicly available datasets, online sources, and published research papers:

- 2,473 CXR images from the PadChest dataset [1]  
- 183 CXR images from a German medical school dataset [2]  
- 559 CXR images from SIRM, GitHub, Kaggle, and Twitter sources [3–6]  
- 400 CXR images from an additional GitHub repository [7]  

---

## Normal Images

A total of 10,192 normal CXR images are collected from the following datasets:

- 8,851 images from RSNA [8]  
- 1,341 images from Kaggle [9]  

---

## Lung Opacity Images

6,012 lung opacity CXR images are collected from the Radiological Society of North America (RSNA) CXR dataset [8].

---

## Viral Pneumonia Images

1,345 viral pneumonia CXR images are collected from the Chest X-Ray Images (Pneumonia) dataset [9].

## Format

All images are provided in **Portable Network Graphics (PNG)** format with a resolution of **299 × 299 pixels**.

---

## Objective

This dataset is intended to support researchers in developing impactful machine learning and deep learning methods for COVID-19 detection using chest X-ray imaging, contributing to improved diagnostic tools during and beyond the pandemic.
